# 100msfireside Monorepo

A turborepo containing:
- **apps/web**: Next.js frontend application
- **apps/backend**: ElysiaJS backend (Bun)

## Getting Started

Install dependencies:
```bash
npm install
```

Run both apps in development:
```bash
npm run dev
```

Build all apps:
```bash
npm run build
```

## Apps

### Web (Next.js)
```bash
cd apps/web
npm run dev
```

### Backend (ElysiaJS)
```bash
cd apps/backend
bun run dev
```
